# CORVUS Web

A futuristic desktop UI built with React, TypeScript, and Tailwind CSS.

## 🚀 How to Run Locally

To see the website on your local browser, follow these steps:

### 1. Install Node.js (If not already installed)
This project requires **Node.js**. If you don't have it, download and install it from [nodejs.org](https://nodejs.org/).

### 2. Open your Terminal or PowerShell
You can do this by right-clicking the `life-os-web` folder and selecting "Open in Terminal".

### 3. Install Dependencies
Type the following command and press Enter:
```bash
npm install
```

### 4. Start the Local Server
Once the installation is finished, run:
```bash
npm run dev
```

### 5. Open in Browser
The terminal will show a link, usually `http://localhost:5173`. **Ctrl + Click** the link to open it in your browser.
